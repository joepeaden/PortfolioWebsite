    Vectors

    Vectors are the most basic object. They only contain elements of the same type.

    Empty vectors can be created with the vector() function.

              > # this is how a vector is created
          > x <- c(0.5, 1.2, 3.2)
          > x
          > [1] 0.5 1.2 3.2
          > # this is how an empty vector is created
          > x <- vector()
